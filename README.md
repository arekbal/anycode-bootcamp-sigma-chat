# AnyCode Bootcamp Sigma Chat

## Backend

```sh
cd ./be/ChatAPI
```

```sh
dotnet run
```

or

```sh
dotnet watch
```

BE starts with `http://localhost:5000`

## Frontend

```sh
cd ./fe
```

```sh
npm install
```

```sh
npm start
```

FE starts with `http://localhost:3000`

Tries to connect to the BE service and writes the response to console (press `F12` in the browser to see).
